# PalindromeWord

A simple application for checking whether **words and sentences are palindromes**.

> A palindrome is a word, phrase, or text that reads the same forward and backward.

## Features

* Simple and clean user interface
* Fast palindrome checking
* Displays the reversed text

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/etet-prog/PalindromeWord.git
cd PalindromeWord
```

### 2. Install Node.js dependencies

```bash
npm install
```

## Usage

### 1. Start the Node.js server

```bash
npm start
```

### 2. Run the Python listener

```bash
cd backend
pip install requests
python server_listener.py
```

### 3. Open the application

Open the following file in your web browser:

```text
frontend/index.html
```

## Notes

* Make sure both the **Node.js** server and the **Python** listener are running simultaneously.
* The application will not work correctly unless both processes are active.
